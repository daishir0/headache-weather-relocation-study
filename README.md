## 📋 プロジェクト概要 / Project Overview

気象条件が頭痛に与える影響を科学的に検証し、**日本全国47都道府県 + ハワイ・ロサンゼルスの計49地域**について、2024年の気象データ（**総計17,605データポイント**）を収集・分析した研究プロジェクトです。

気圧変動・湿度・日照時間の3つの指標を統合評価し、**頭痛患者に最適な移住先**を科学的エビデンスに基づいてランキング化しました。

### Key Features

- **49地域の包括的分析**: 日本全国47都道府県 + 国際2都市（ハワイ、ロサンゼルス）
- **17,605データポイント**: 2024年の実測気象データを使用（仮説・推測なし）
- **3指標統合評価**: 気圧変動（40%）、湿度（30%）、日照時間（30%）の重み付けスコアリング
- **統計的検証**: Pearson相関分析・t検定による有意性検証
- **学術論文・ブログ記事**: 研究成果を学術論文とブログ記事の両形式で公開

## 🎯 研究の背景と目的 / Background & Objectives

### なぜこの研究が必要か？

気象条件と頭痛の関係は医学的にも広く認められており、特に以下の3つの要因が頭痛の引き金になることが多くの研究で報告されています：

1. **気圧変動**: 気圧が急激に低下すると、内耳の気圧センサーが刺激され、脳の三叉神経が活性化されて頭痛が発生
2. **高湿度**: 体温調節がうまくいかず、血管が拡張して頭痛を引き起こす
3. **日照不足**: セロトニン不足により、痛みを感じやすくなる

しかし、これまで**3つの指標を統合的に評価し、地域間を比較した包括的研究**は存在しませんでした。

### 研究目的

本研究は、**気圧・湿度・日照の3指標を統合評価**することで、**頭痛患者に最適な移住先**を科学的に特定することを目的としています。

### 統計的知見

#### 1️⃣ 日照時間が最も重要（相関係数 r=0.806, p<0.001）

日照時間と総合スコアの間に**極めて強い正の相関**が認められました。日照時間が長い地域ほど、頭痛患者にとって住みやすい環境であることが統計的に証明されました。

#### 2️⃣ 湿度の影響も無視できない（p<0.0001）

上位10地域と下位10地域を比較したところ、**湿度差が最も統計的に有意**でした：
- 上位10地域の平均湿度: **66.6%**
- 下位10地域の平均湿度: **75.2%**
- 差: 約9% (p<0.0001)

#### 3️⃣ 気圧変動も統計的に有意（p=0.016）

上位10地域の平均気圧日較差は**4.85hPa**、下位10地域は**6.04hPa**で、統計的に有意な差が認められました。

### データ処理手法

#### 気圧データ
- 気象庁の1時間ごとの気圧データから、1日の最高気圧と最低気圧の差（日較差）を算出
- 日較差が小さいほど、気圧変動による頭痛リスクが低い

#### 湿度データ
- 国内: 気象庁の1時間ごとの相対湿度から日平均値を算出
- 国際: NOAAの気温・露点温度データから**Magnus式**で相対湿度を計算
  ```
  e = 6.112 × exp((17.67 × dewpoint) / (dewpoint + 243.5))
  es = 6.112 × exp((17.67 × temp) / (temp + 243.5))
  RH = (e / es) × 100
  ```

#### 日照データ
- 国内: 気象庁の日照計で実測された日照時間
- 国際: NOAAの雲量データから推定

### スコアリング手法

各指標を0～100点のスケールに正規化し、以下の重み付けで総合スコアを算出：

```
総合スコア = 気圧スコア × 0.40 + 湿度スコア × 0.30 + 日照スコア × 0.30
```

- **気圧日較差**: 低いほど良い（逆スコア）
- **湿度**: 低いほど良い（逆スコア）
- **日照時間**: 長いほど良い（正スコア）

### 1. リポジトリのクローン

```bash

cd headache-weather-relocation-study
```

### 2. 環境構築

```bash
# Python環境（推奨: Python 3.11以上）

```

